# WebPwd 切换账号下拉与当前账号标识上线操作说明

## 配置项说明
当前改动不新增配置项。

## 本地检查命令
```powershell
Set-Location d:\project\google-extension
Get-Item .\web_pwd\content_script.js
```

## Chrome 扩展刷新步骤
1. 打开 `chrome://extensions/`
2. 开启“开发者模式”
3. 在 `web_pwd` 扩展上点击“重新加载”
4. 刷新目标登录页后重新验证

## 建议验证步骤
```powershell
Set-Location d:\project\google-extension
Write-Output '1. 重新加载 web_pwd 扩展'
Write-Output '2. 在同一登录页保存两个及以上账号'
Write-Output '3. 确认右上角显示切换账号按钮'
Write-Output '4. 点击后以下拉形式展示当前登录页全部账号'
Write-Output '5. 确认点击切换账号按钮后，下拉不会立即消失'
Write-Output '6. 确认默认填充账号带有当前标识'
Write-Output '7. 切换其他账号后确认当前标识跟随更新'
Write-Output '8. 点击登录后确认切换账号入口消失'
```

## 打包命令
```powershell
Set-Location d:\project\google-extension
Compress-Archive -Path .\web_pwd\* -DestinationPath .\web_pwd-account-switch-dropdown-current.zip -Force
```

## 上线风险提示
- 下拉入口为页面注入 UI，建议验证常规表单登录和 Ajax 登录场景。
- 建议验证多账号数量较多时的下拉展示与当前标识是否仍清晰可用。
