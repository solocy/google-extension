# WebPwd 页面内滑出保存卡片上线操作说明

## 配置项说明
当前改动无需新增配置项。

## 本地验证
```powershell
# 进入项目根目录
Set-Location d:\project\google-extension

# 检查目标文件是否存在
Get-Item .\web_pwd\content_script.js
```

## Chrome 扩展加载/刷新
1. 打开 `chrome://extensions/`
2. 打开“开发者模式”
3. 在已加载的 `web_pwd` 扩展卡片上点击“重新加载”
4. 刷新目标登录页面后重新提交登录表单验证

## 建议验证步骤
```powershell
# 无自动化构建，当前以浏览器手工验证为主
Set-Location d:\project\google-extension
Write-Output '1. 重新加载扩展'
Write-Output '2. 打开任意登录页并提交账号密码'
Write-Output '3. 检查是否右下角滑入卡片并在10秒后消失'
```

## 打包命令
如需归档当前扩展目录，可执行：
```powershell
Set-Location d:\project\google-extension
Compress-Archive -Path .\web_pwd\* -DestinationPath .\web_pwd-inline-save-card.zip -Force
```

## 上线风险提示
- 当前主要为前端交互改造，风险集中在页面注入样式与登录页兼容性。
- 建议至少验证一个标准表单登录页和一个 Ajax 登录页。
