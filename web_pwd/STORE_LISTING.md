# WebPwd - Chrome Web Store 上线文档

## 一、扩展基本信息

### 名称
**WebPwd - 密码管理器** / WebPwd - Password Manager

### 简短描述 (132字符以内)
安全记录网页登录信息，支持自动填充、文件夹管理，所有数据本地存储加密。

### 详细描述

```
WebPwd 是一款轻量级的浏览器密码管理器扩展，帮助您安全、便捷地管理网站登录信息。

主要功能：
🔐 自动检测并保存登录信息 - 当您在网站输入用户名和密码时，WebPwd 会智能识别并提示保存
🚀 一键自动填充 - 访问已保存的网站时，自动填充登录信息，省去重复输入的麻烦
📁 文件夹管理 - 将密码分类整理到不同文件夹，方便查找和管理
🔒 本地存储 - 所有数据仅存储在您的本地浏览器中，不会上传到任何服务器
🛡��� 安全加密 - 敏感信息采用加密存储，保护您的隐私安全

适用场景：
• 需要管理多个网站账号的用户
• 经常忘记密码的用户
• 注重隐私安全，不想使用云端密码管理器的用户
• 希望快速登录常用网站的用户

使用方法：
1. 安装扩展后，正常访问需要登录的网站
2. 输入用户名和密码后，会出现保存提示
3. 选择保存文件夹并确认
4. 下次访问该网站时，登录信息将自动填充

注意事项：
• 请定期备份您的密码数据
• 建议设置浏览器主密码以增强安全性
• 数据仅存储在当前浏览器，卸载扩展或清除浏览器数据���导致密码丢失

WebPwd - 让密码管理更简单、更安全！
```

## 四、权限声明（审核用）

### 权限使用理由说明

| 权限 | 用途说明 |
|------|----------|
| `storage` | 用于在本地存储用户保存的密码凭证、文件夹配置和设置选项。所有数据仅存储在用户本地浏览器中，不会上传到任何服务器。 |
| `tabs` | 用于获取当前标签页的 URL，以便匹��已保存的凭证并触发自���填充功能。 |
| `<all_urls>` | 用于在所有网站上注入内容脚本，检测登录表单并提供密码保存和自动填充功能。这是密码管理器的核心功能所必需的。 |

### 远程代码声明
本扩展不加载或执行任何远程代码。所有 JavaScript 代码都打包在扩展内部。

## 六、发布流程

### 步骤 1：注册开发者账号
1. 访问 [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. 使用 Google 账号登录
3. 支付一次性 $5 注册费（如未注册过）

### 步骤 2：打包扩展
```powershell
# 在扩展目录中创建 zip 文件
cd D:\project\google-extension\web_pwd
Compress-Archive -Path * -DestinationPath webpwd.zip
```

### 步骤 3：上传扩展
1. 在 Developer Dashboard 点击 "新商品"
2. 上传 webpwd.zip 文件
3. 填写商店列表信息
4. 上传图标和截图
5. 填写隐私政策 URL
6. 提交审核

### 步骤 4：审核等待
- 通常需要 1-3 个工作日
- 首次提交可能需要更长时间
- 如被拒绝，根据反馈修改后重新提交
