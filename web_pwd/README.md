# WebPwd - 网页登录密码管理扩展

一个简洁高效的 Chrome 扩展，用于保存和自动填充网页登录凭证，支持文件夹分类管理，类似书签的方式组织登录信息。

## 核心功能

### 1. 自动检测并保存登录信息
- ✅ 检测网页登录表单提交
- ✅ 智能识别用户名和密码输入框
- ✅ 弹出保存提示，支持选择文件夹
- ✅ 可自定义登录信息标题

### 2. 自动填充登录
- ✅ 进入已保存的登录页面时自动填充用户名密码
- ✅ 智能匹配算法，根据URL和页面特征找到最佳凭证
- ✅ 支持开启/关闭自动填充功能

### 3. 文件夹管理（类似书签）
- ✅ 创建多个文件夹分类管理登录信息
- ✅ 保存时选择文件夹
- ✅ 按文件夹分组显示凭证列表

### 4. 从保存的书签打开并登录
- ✅ 点击"打开并登录"按钮直接打开网站并自动填充
- ✅ 支持"填充"、"填充并登录"、"打开并登录"三种模式
- ✅ 自动提交表单（可选）

## 文件结构

```
web_pwd/

├── background.js          # 后台服务 worker，管理数据存储和消息路由

├── popup.html            # 弹出窗口 UI
├── popup.js              # 弹出窗口逻辑，管理凭证列表
├── styles.css            # UI 样式
├── crypto.js             # 加密辅助（占位，未启用）
└── README.md             # 本文档
```

## 安装步骤

1. 打开 Chrome 浏览器，进入扩展管理页面：
   - 地址栏输入：`chrome://extensions/`
   - 或菜单：更多工具 → 扩展程序

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择 `web_pwd` 目录

5. 扩展图标会显示在工具栏（可固定）

## 使用方法

### 保存登录信息

1. 访问任意登录页面（如 GitHub、Gmail 等）
2. 输入用户名和密码，点击登录
3. 浏览器会打开一个独立的扩展保存弹窗（持续 5 秒）
4. 选择要保存到的文件夹
5. 可选：修改标题（默认为网站域名）
6. 点击"保存"按钮

### 自动填充登录

1. 默认启用自动填充（可在popup中切换）
2. 再次访问已保存的登录页面
3. 页面加载完成后自动填充用户名和密码
4. 手动点击登录按钮即可

### 手动填充

1. 访问登录页面
2. 点击扩展图标打开popup
3. 找到对应的登录信息
4. 点击"填充"或"填充并登录"按钮

### 从书签打开

1. 点击扩展图标打开popup
2. 浏览保存的登录信息（按文件夹分组）
3. 点击"打开并登录"按钮
4. 扩展会打开新标签页并自动填充密码、自动登录

### 管理文件夹

1. 打开popup
2. 点击"新建文件夹"按钮
3. 输入文件夹名称
4. 新保存的凭证可以选择该文件夹

### 删除凭证

1. 打开popup
2. 找到要删除的凭证
3. 点击"删除"按钮
4. 确认删除

## 技术特性

### 智能匹配算法

扩展使用评分系统匹配最佳凭证：
- Origin 完全匹配（+50分）
- URL 完全匹配（+40分）
- 同域名不同路径（+20分）
- 路径前缀匹配（+10分）
- 表单选择器匹配（+5分）
- 输入框选择器匹配（+3分）

### 数据模型

**凭证（Credential）：**
- id: 唯一标识
- title: 显示标题
- username: 用户名
- password: 密码（明文存储，原型版本）
- origin: 网站源（如 https://github.com）
- urlPattern: 完整URL
- formSelector: 表单CSS选择器
- usernameSelector: 用户名输入框选择器
- passwordSelector: 密码输入框选择器
- tags: 标签数组（预留）
- folderId: 所属文件夹ID
- createdAt: 创建时间
- updatedAt: 更新时间

**文件夹（Folder）：**
- id: 唯一标识
- name: 文件夹名称
- color: 颜色（预留）
- createdAt: 创建时间

## 安全说明

⚠️ **重要安全提示：**

1. **本版本为原型，密码以明文形式存储在 chrome.storage.local 中**
2. 仅适合开发测试或保存不重要的账号
3. 请勿在生产环境或保存敏感账户时使用

### 后续安全改进建议：

1. 实现主密码功能：
   - 使用 PBKDF2 派生密钥

3. 添加导入/导出功能（加密备份）

## 故障排除

**问题：保存提示没有出现**
- 检查页面是否有 `<input type="password">` 字段
- 确认扩展已启用且有权限访问该网站

**问题：自动填充不工作**
- 检查popup中"自动填充"开关是否已启用
- 确认凭证的 origin 与当前页面匹配
- 某些SPA页面可能需要延迟加载，可手动填充

**问题：填充后无法登录**
- 某些网站有额外的验证（验证码、2FA等）
- 某些网站的登录逻辑可能不支持自动提交
- 可尝试填充后手动点击登录按钮
