## 技术实现

### 前端技术
- **content_script.js** - 网页脚本，检测登录并管理UI
- **popup.js** - 扩展弹窗，管理凭证和文件夹
- **styles.css** - 现代化UI设计

### 后端技术
- **background.js** - Service Worker，处理数据和消息
- **manifest.json** - 扩展配置，权限管理

### 数据存储
- **chrome.storage.local** - 本地存储凭证
- **自动初始化** - 首次使用自动创建默认文件夹

## 安装和使用

### 1. 安装
```
1. chrome://extensions/
2. 开启开发者模式
3. 加载已解压的扩展程序
4. 选择 web_pwd 文件夹
```

### 2. 基本使用
```
1. 访问登录页面
2. 输入用户名和密码
3. 点击登录
4. 弹窗提示保存
5. 选择文件夹并保存
```

### 3. 自动填充
```
1. 再次访问同一页面
2. 自动填充用户名密码
3. 可在popup关闭此功能
```

### 4. 从书签登录
```
1. 打开扩展弹窗
2. 点击"打开并登录"
3. 自动打开和登录
```
