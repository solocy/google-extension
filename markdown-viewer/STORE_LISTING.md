# Chrome Web Store 商店信息

## 基本信息

### 扩展名称
**英文**: Markdown Viewer
**中文**: Markdown查看器

### 简短描述（132字符以内）
**英文**: Elegant Markdown file viewer with GitHub-style rendering, view toggle, and quick copy features.

**中文**: 优雅的Markdown文件查看器，GitHub风格渲染，支持视图切换和快速复制功能。

### 详细描述

**英文版本**:
```
Markdown Viewer is a lightweight Chrome extension designed for developers and documentation writers. It automatically detects and renders .md or .markdown files into beautifully formatted documents.

KEY FEATURES:

🎨 GitHub-Style Rendering
• Standard GitHub Markdown styling
• Full Markdown syntax support
• Code block syntax highlighting
• Perfect rendering of tables, lists, and quotes

📝 Flexible View Switching
• Toggle between rendered and raw text views
• Easy comparison of Markdown source
• Seamless reading and editing experience

📋 Convenient Content Operations
• One-click copy of Markdown source
• Keyboard shortcut support
• Responsive design for all screen sizes

🌐 Wide Compatibility
• Local file access support
• Network URL Markdown files
• Compatible with various Markdown dialects

🚀 Lightweight & Efficient
• No external dependencies
• Instant rendering with no delay
• Minimal resource usage

PERFECT FOR:
• Viewing GitHub repository README files
• Reading technical and API documentation
• Previewing local Markdown notes
• Reviewing Markdown articles
• Learning Markdown syntax

PRIVACY:
• No data collection
• All processing done locally
• No external server communication
• Open source and transparent

PERMISSIONS:
• storage: Save user preferences locally
• file:///*: Access local Markdown files you open
• http/https: Render network Markdown files you visit

Get started in seconds - just install and open any .md file!
```

**中文版本**:

## 提交注意事项

### 审核要点
1. **功能完整性**: 确保所有描述的功能都能正常工作
2. **权限合理性**: 只请求必要的权限，并清楚说明用途
3. **隐私合规**: 明确声明数据处理方式
4. **用户体验**: 界面友好，操作直观
5. **性能表现**: 不影响浏览器性能

### 常见拒绝原因
- 权限请求过多或说明不清
- 功能描述与实际不符
- 隐私政策缺失或不完整
- 存在安全漏洞
- 侵犯知识产权
- 用户体验差

### 审核时间
- 首次提交：通常1-3个工作日
- 更新版本：通常1-2个工作日
- 如被拒绝：修改后可立即重新提交
