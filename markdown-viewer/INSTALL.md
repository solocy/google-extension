# 安装指南

## 快速开始

### 1. 加载扩展

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角开启"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `markdown-viewer` 文件夹

### 2. 启用文件访问（重要）

要查看本地的.md文件，必须启用文件访问权限：

1. 在扩展列表中找到"Markdown Viewer"
2. 点击"详细信息"
3. 向下滚动找到"允许访问文件网址"选项
4. 开启该选项

### 3. 使用扩展

现在你可以：
- 直接在Chrome中打开任何 `.md` 或 `.markdown` 文件
- 扩展会自动检测并渲染Markdown内容
- 使用工具栏按钮切换原文视图或复制内容

## 测试扩展

创建一个测试文件 `test.md`：

```markdown
# 测试标题

这是一个**粗体**文本和*斜体*文本。

## 代码示例

\`\`\`javascript
console.log('Hello, Markdown!');
\`\`\`

## 列表

- 项目 1
- 项目 2
- 项目 3

## 链接

[访问GitHub](https://github.com)
```

然后在Chrome中打开这个文件，你应该能看到格式化的渲染结果。

## 故障排除

### 扩展没有生效

- 确认已启用"允许访问文件网址"
- 检查文件扩展名是否为 `.md` 或 `.markdown`
- 尝试刷新页面（F5）

### 样式显示不正常

- 清除浏览器缓存
- 重新加载扩展

### 无法访问网络上的Markdown文件

- 某些网站可能有内容安全策略限制
- 建议下载文件到本地查看
