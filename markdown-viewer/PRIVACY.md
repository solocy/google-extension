## 中文版

### 1. 概述

Markdown Viewer（以下简称"本扩展"）尊重并保护用户隐私。本隐私政策说明了本扩展如何处理用户信息。

### 2. 信息收集

**本扩展不收集任何用户信息。**

具体而言，本扩展不会收集、存储或传输以下任何信息：
- 个人身份信息（姓名、邮箱、电话等）
- 浏览历史记录
- 打开的文件内容
- 文件路径或文件名
- IP地址
- 设备信息
- 使用统计数据
- 任何其他个人或非个人信息

### 3. 数据存储

本扩展仅在用户本地浏览器中存储以下设置：
- 用户界面偏好设置（如主题选择）
- 扩展配置选项

这些数据：
- 仅存储在用户本地设备
- 不会上传到任何服务器
- 不会与任何第三方共享
- 用户可随时通过卸载扩展来删除

### 4. 权限使用说明

本扩展请求以下权限，仅用于核心功能：

#### 4.1 storage权限
- **用途**: 在本地保存用户偏好设置
- **数据范围**: 仅限界面配置选项
- **存储位置**: 用户本地浏览器

#### 4.2 file:///* 权限
- **用途**: 读取用户主动打开的本地Markdown文件
- **数据范围**: 仅限用户明确选择打开的.md文件
- **处理方式**: 仅在浏览器内渲染，不上传或存储

#### 4.3 http://*/* 和 https://*/* 权限
- **用途**: 渲染用户访问的网络Markdown文件
- **数据范围**: 仅限用户主动访问的URL
- **处理方式**: 仅在浏览器内渲染，不上传或存储

### 5. 第三方服务

**本扩展不使用任何第三方服务**，包括但不限于：
- 分析服务（如Google Analytics）
- 广告服务
- 云存储服务
- API服务
- 任何外部服务器

### 6. 数据传输

**本扩展不进行任何网络通信**，除了：
- 用户主动访问的网络Markdown文件的正常HTTP/HTTPS请求
- 这些请求由浏览器直接处理，扩展不拦截或修改

### 7. Cookies

本扩展不使用Cookies或任何类似的跟踪技术。

### 8. 儿童隐私

本扩展不针对13岁以下儿童，也不会故意收集儿童的个人信息。

### 9. 数据安全

由于本扩展不收集或传输任何数据，因此不存在数据泄露风险。所有处理均在用户本地设备上完成。

### 10. 用户权利

用户拥有以下权利：
- **访问权**: 查看本地存储的设置数据
- **删除权**: 通过卸载扩展删除所有本地数据
- **控制权**: 完全控制扩展的权限和使用

### 11. 隐私政策变更

如本隐私政策发生变更，我们会：
- 更新本文档的"最后更新日期"
- 在扩展更新说明中通知用户
- 重大变更会在扩展中显著提示

### 12. 开源透明

本扩展是开源项目，源代码公开可查：
- 用户可审查所有代码
- 确认不存在隐藏的数据收集行为
- 欢迎社区监督和审计

### 13. 联系方式

如对本隐私政策有任何疑问，请通过以下方式联系：
- GitHub Issues: [项目地址]
- 邮箱: [联系邮箱]

## English Version

### 1. Overview

Markdown Viewer (hereinafter referred to as "the Extension") respects and protects user privacy. This Privacy Policy explains how the Extension handles user information.

### 2. Information Collection

**The Extension does NOT collect any user information.**

Specifically, the Extension does not collect, store, or transmit any of the following:
- Personal identification information (name, email, phone, etc.)
- Browsing history
- File contents
- File paths or filenames
- IP addresses
- Device information
- Usage statistics
- Any other personal or non-personal information

### 3. Data Storage

The Extension only stores the following settings locally in the user's browser:
- User interface preferences (such as theme selection)
- Extension configuration options

This data:
- Is stored only on the user's local device
- Is not uploaded to any server
- Is not shared with any third parties
- Can be deleted at any time by uninstalling the Extension

### 4. Permission Usage

The Extension requests the following permissions for core functionality only:

#### 4.1 storage Permission
- **Purpose**: Save user preferences locally
- **Data Scope**: Interface configuration options only
- **Storage Location**: User's local browser

#### 4.2 file:///* Permission
- **Purpose**: Read local Markdown files that users actively open
- **Data Scope**: Only .md files explicitly selected by users
- **Processing**: Rendered in browser only, not uploaded or stored

#### 4.3 http://*/* and https://*/* Permissions
- **Purpose**: Render network Markdown files that users visit
- **Data Scope**: Only URLs actively accessed by users
- **Processing**: Rendered in browser only, not uploaded or stored

### 5. Third-Party Services

**The Extension does NOT use any third-party services**, including but not limited to:
- Analytics services (e.g., Google Analytics)
- Advertising services
- Cloud storage services
- API services
- Any external servers

### 6. Data Transmission

**The Extension does NOT perform any network communication**, except:
- Normal HTTP/HTTPS requests for network Markdown files that users actively visit
- These requests are handled directly by the browser; the Extension does not intercept or modify them

### 7. Cookies

The Extension does not use cookies or any similar tracking technologies.

### 8. Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect personal information from children.

### 9. Data Security

Since the Extension does not collect or transmit any data, there is no risk of data breach. All processing is done locally on the user's device.

### 10. User Rights

Users have the following rights:
- **Access**: View locally stored settings data
- **Deletion**: Delete all local data by uninstalling the Extension
- **Control**: Full control over the Extension's permissions and usage

### 11. Privacy Policy Changes

If this Privacy Policy changes, we will:
- Update the "Last Updated" date in this document
- Notify users in the extension update notes
- Prominently notify users of significant changes within the Extension

### 12. Open Source Transparency

The Extension is an open-source project with publicly available source code:
- Users can review all code
- Confirm there is no hidden data collection
- Community oversight and auditing are welcome

### 13. Contact Information

If you have any questions about this Privacy Policy, please contact us:
- GitHub Issues: [Project URL]
- Email: [Contact Email]
