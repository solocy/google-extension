# QR Code Scanner

A lightweight Chrome extension for scanning and decoding QR codes on any webpage.

## Features

- **Instant Scanning** - Click the extension icon to scan QR codes on the current page
- **Multiple QR Codes** - Detects and lists all visible QR codes
- **One-Click Copy** - Easily copy decoded content to clipboard
- **Smart Detection** - Identifies URLs, emails, phone numbers, WiFi configs
- **Privacy-First** - All processing happens locally, no data sent to servers
- **Minimal Permissions** - Only requires `activeTab` permission

### Manual Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the `qrcode` folder

## Usage

1. Navigate to a webpage with QR codes
2. Click the QR Code Scanner icon in the toolbar
3. Select a QR code from the list
4. Click the copy button to copy the content

## Permissions

This extension only requires:
- `activeTab` - To capture the visible tab when you click the icon

## Privacy

- No data collection
- No external server communication
- All processing done locally in your browser
