# QR Code Scanner - 二维码扫描器

一个轻量级的 Chrome 扩展，用于扫描和解码网页上的二维码。

## 功能特点

- **即时扫描** - 点击扩展图标即可扫描当前页面的二维码
- **多码识别** - 检测并列出页面上所有可见的二维码
- **一键复制** - 轻松复制解码后的内容到剪贴板
- **智能识别** - 自动识别网址、邮箱、电话号码、WiFi配置等
- **隐私优先** - 所有处理在本地完成，不向服务器发送任何数据
- **权限精简** - 仅需要 `activeTab` 权限

## 安装方法

### 从 Chrome 网上应用店安装
*（即将上线）*

### 手动安装
1. 下载或克隆此仓库
2. 打开 Chrome，访问 `chrome://extensions/`
3. 在右上角启用"开发者模式"
4. 点击"加载已解压的扩展程序"，选择 `qrcode` 文件夹

## 使用方法

1. 打开包含二维码的网页
2. 点击工具栏中的二维码扫描器图标
3. 从列表中选择一个二维码
4. 点击复制按钮复制内容

## 权限说明

本扩展仅需要：
- `activeTab` - 用于在点击图标时截取当前标签页

## 隐私声明

- 不收集任何数据
- 不与外部服务器通信
- 所有处理都在浏览器本地完成
